# 메신저 프로그램

## 프로젝트 개요
이 프로젝트는 Cpp/CLI와 소켓통신을 이용한 간단한 메신저 프로그램입니다.
회원과 채팅기록을 관리하기 위하여 MySQL를 사용하였으며, 클라이언트와 서버를 만들어 연동하게 하였습니다.
